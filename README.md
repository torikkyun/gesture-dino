# Gesture Dino

## Cài đặt dự án

```bash
uv sync
```

## Cách sử dụng

### Bước 1. Mở game Dino

1. Mở trình duyệt
2. Vào trang [Dino Game](https://chromedino.com/)

### Bước 2. Chạy chương trình

```bash
uv run main.py
```

### Bước 3. Điều khiển game

Sử dụng các cử chỉ:

- ✌️: Nhảy (Space)
- Nắm tay (0 ngón): Không nhảy
